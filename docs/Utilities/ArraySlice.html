static void slice( array $array, int|null $start = null, int|null $stop = null, int|null $step = null, bool $byReference = false)

Implements the Python slice behaviour

  • a[1:2:3] => slice($a, 1, 2, 3)
  • a[1:4] => slice($a, 1, 4)
  • a[3::2] => slice($a, 3, null, 2)

If $byReference is true, then the elements of the resulting array will be references to the initial array.
